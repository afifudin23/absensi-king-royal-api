# Absensi King Royal API

Struktur awal project Go (Gin) untuk skala menengah.

## Folder Overview

- `cmd/api`: entry point aplikasi (main server).
- `configs`: file konfigurasi per environment.
- `deployments`: aset untuk deployment (Docker, dll).
- `docs`: dokumentasi project dan API.
- `internal`: business logic dan implementasi internal app.
- `migrations`: file migrasi database.
- `pkg`: utility reusable lintas package.
- `scripts`: script bantu development/ops.
- `test`: test terpisah dari source utama.
